* The following symbols are hard-coded, i.e. they should not changed
  FOL equality    : ≡
  FOL conjunction : ∧
  FOL disjunction : ∨
  FOL negation    : ¬
  FOL false       : ⊥
  FOL existential : ∃D

* Agda version
  The agda2atp must be compiled using the same version of Agda than
  the used to generate the Agda interface files.
