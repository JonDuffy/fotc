------------------------------------------------------------------------------
Axiomatic Peano arithmetic
------------------------------------------------------------------------------

This directory contains the theory of Peano arithmetic, using Agda
postulates for the non-logical constants and the Peano's axioms (see,
for example, [p. 263, 1], [p. 28, 2])).

[1] Moshé Machover. Set theory, logic and their limitations. Cambridge
    University Press, 1996.

[2] Petr Hájeck and Pavel Pudlák. Metamathematics of First-Order
    Arithmetic. Springer, 1998. 2nd printing.
