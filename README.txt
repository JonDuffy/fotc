==============================================================================
FOT (First Order Theories)
==============================================================================

  Code accompanying the paper "Combining Automatic and Interactive
  Proof in First Order Theories of Combinators" by Ana Bove, Peter
  Dybjer, and Andrés Sicard-Ramírez.

------------------------------------------------------------------------------
Description
------------------------------------------------------------------------------

  Examples of the formalization of first order theories showing the
  combination of interactive proofs with automatics proofs carried out
  by first order automatic theorem provers (ATPs).

------------------------------------------------------------------------------
Information
------------------------------------------------------------------------------

For information about the FOT code, see README.agda.
